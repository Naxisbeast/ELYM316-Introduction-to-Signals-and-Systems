# Lab 3: Fast Fourier Transforms

## Overview

This lab generated 1 Hz, 5 Hz, and 10 Hz sinusoidal signals, combined them, added noise, and used Fast Fourier Transform analysis to recover the original frequency components and their amplitudes.

## Concepts Applied

- Sinusoidal signal generation
- Superposition
- Additive noise
- Time-domain and frequency-domain representations
- FFT magnitude spectra
- Frequency and amplitude recovery
