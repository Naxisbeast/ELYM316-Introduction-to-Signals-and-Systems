# Lab 4: Radioactive Decay

## Overview

This lab simulated radioactive decay using the Euler algorithm, estimated the half-life and time constant from numerical data, and compared the results with theoretical expectations for exponential decay.

## Concepts Applied

- First-order differential equations
- Exponential decay
- Euler numerical integration
- Half-life and time constant
- Numerical error and step size
