# ELYM316 – Introduction to Signals and Systems

## Module Overview

ELYM316 introduced continuous-time and discrete-time signals, linear time-invariant (LTI) systems, Fourier and Laplace transform techniques, filtering, sampling, modulation, reconstruction, and MATLAB-based signal analysis. The module connected mathematical descriptions of signals with practical modelling, visualisation, and interpretation.

## Portfolio Purpose

This repository documents my laboratory work, MATLAB simulations, signal-analysis scripts, verified laboratory performance, and reflections from the module. It is organised as an academic and technical portfolio so that each item has context, reproducible code, and a clear connection to the concepts studied.
